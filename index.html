Sección 1

¿Qué es HTML y CSS?

🏗️ HTML — La estructura

HTML (HyperText Markup Language) es el esqueleto de toda página web. Define el contenido: títulos, párrafos, imágenes, botones, etc. Sin HTML, no hay página web.

🎨 CSS — El estilo

CSS (Cascading Style Sheets) es el maquillaje de la página web. Define colores, tamaños, fuentes, espacios... Convierte una página aburrida en algo bonito.

💡 Analogía del día Imagina una casa: HTML es la estructura (paredes, puertas, ventanas) y CSS es la decoración (color de las paredes, muebles, iluminación). Juntos hacen que la casa sea habitable y hermosa.

¿Cómo funciona una página web?

1

Tú escribes código HTML y CSS

En un editor de texto creas los archivos .html y .css

2

El navegador lee tu código

Chrome, Firefox o Edge interpretan el código y lo convierten en la página visual

3

¡El usuario ve tu página!

Cualquier persona puede abrir tu archivo en su navegador y ver el resultado

Antes de empezar

Herramientas necesarias

📝 Editor de código

Necesitas un editor para escribir tu código. El más popular y gratuito es Visual Studio Code (VS Code). Descárgalo en code.visualstudio.com

🌐 Navegador web

Para ver el resultado de tu código. Usa Google Chrome o Firefox. Ambos son gratuitos y tienen herramientas para desarrolladores.

🛠️ Opcional: Live Server

Es una extensión de VS Code que actualiza la página automáticamente cuando guardas. Búscala en las extensiones de VS Code con el nombre Live Server.

✅ Alternativa online gratuita Si no puedes instalar programas, usa CodePen (codepen.io) o Replit (replit.com). Son editores web donde puedes practicar HTML y CSS directamente en el navegador.

Sección 2

Mi primer archivo HTML

Todo archivo HTML empieza con la misma estructura base. Es como el "molde" que siempre debes usar.

📄 mi-primera-pagina.html 
<!-- Esta línea le dice al navegador que es HTML5 -->
<!DOCTYPE html>

<!-- El idioma de la página (es = español) -->
<html lang="es">

  <head>
    <!-- El tipo de caracteres (permite tildes y ñ) -->
    <meta charset="UTF-8">
    <!-- Para que se vea bien en celulares -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- El título que aparece en la pestaña del navegador -->
    <title>Mi primera página</title>
  </head>

  <body>
    <!-- Todo lo visible va aquí -->
    <h1>¡Hola, mundo!</h1>
    <p>Esta es mi primera página web.</p>
  </body>

</html>

🧠 <head> — La cabeza

Contiene información para el navegador: título de la página, codificación de caracteres, enlace al CSS. El usuario NO ve este contenido directamente.

👁️ <body> — El cuerpo

Todo lo que el usuario ve en pantalla va aquí: textos, imágenes, botones, listas. Este es el área de trabajo principal.

⚠️ Regla importante Las etiquetas HTML son como paréntesis: siempre deben abrirse y cerrarse. Si abres <p>, debes cerrarlo con </p>. La barra inclinada "/" indica que es el cierre.
Ejercicio 1

¡Tu primera página!

  1. Abre VS Code y crea un nuevo archivo llamado index.html
  2. Copia la estructura base de arriba
  3. Cambia el título a tu nombre
  4. En el <body>, escribe: <h1>Hola, soy [tu nombre]</h1>
  5. Guarda el archivo (Ctrl+S) y ábrelo con el navegador

🎯 Objetivo: Ver tu nombre como título en el navegador.

Sección 3

Etiquetas fundamentales

Las etiquetas son los bloques básicos de HTML. Cada etiqueta tiene un propósito específico.

Etiqueta Para qué sirve Ejemplo
<h1> ... <h6> Títulos (h1 = más grande, h6 = más pequeño) <h1>Título principal</h1>
<p> Párrafo de texto <p>Mi texto aquí</p>
<strong> Texto en negrita <strong>Importante</strong>
<em> Texto en cursiva <em>Énfasis</em>
<br> Salto de línea (no tiene cierre) Línea 1<br>Línea 2
<hr> Línea horizontal divisoria <hr>
<div> Contenedor genérico (bloque) <div>...</div>
<span> Contenedor en línea (para texto) <span>palabra</span>

Los títulos H1 al H6

Vista previa en el navegador:

H1 — Título principal

H2 — Subtítulo

H3 — Sección

H4 — Subsección

H5 — Detalle
H6 — El más pequeño
💡 Buena práctica Usa solo un <h1> por página (el título principal). Luego puedes usar h2, h3, etc. para organizar el contenido. Es como los capítulos de un libro.

Sección 4

Listas

📌 Lista sin orden (ul)

HTML
<ul>
  <li>Manzana</li>
  <li>Banana</li>
  <li>Naranja</li>
</ul>
Resultado:
  • Manzana
  • Banana
  • Naranja

🔢 Lista ordenada (ol)

HTML
<ol>
  <li>Despertar</li>
  <li>Desayunar</li>
  <li>Ir al colegio</li>
</ol>
Resultado:
  1. Despertar
  2. Desayunar
  3. Ir al colegio

Lista de tus materias favoritas

  1. Crea una lista ordenada con tus 3 materias favoritas del colegio
  2. Crea una lista sin orden con 4 hobbies o pasatiempos
  3. Añade un título h2 antes de cada lista
Solución posible
<h2>Mis materias favoritas</h2>
<ol>
  <li>Matemáticas</li>
  <li>Ciencias</li>
  <li>Arte</li>
</ol>

<h2>Mis hobbies</h2>

  <li>Leer</li>
  <li>Fútbol</li>
  <li>Escuchar música</li>

</ul>

Sección 5

Enlaces e imágenes

🔗 La etiqueta <a> — Hipervínculos

La etiqueta <a> (anchor) crea enlaces que nos llevan a otras páginas.

HTML — Tipos de enlaces
<!-- Enlace a otra página web -->
<a href="https://www.google.com">Ir a Google</a>

<!-- Enlace que abre en pestaña nueva -->
<a href="https://www.google.com" target="_blank">Ir a Google en pestaña nueva</a>

<!-- Enlace a otra página de TU sitio -->
<a href="contacto.html">Ir a Contacto</a>

<!-- Enlace para enviar un correo -->
<a href="mailto:tu@email.com">Enviarme un email</a>
